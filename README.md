# my-cli
![Npm 版本](https://img.shields.io/badge/my--cli-v0.0.1-green)

## 简介
- 用于快速搭建前端项目的命令行工具

## 模版
- 快速生成Vue3-ElementPlus-Vite-TypeScript的模版
- 快速生成Vue2-ElementUi的模版

## 安装

```bash
# 使用模版创建新项目
my-cli create

# 执行项目文件夹名称并选择模版创建新项目
my-cli create [projectName]

# 查看脚手架版本
my-cli -v
my-cli -version

```
